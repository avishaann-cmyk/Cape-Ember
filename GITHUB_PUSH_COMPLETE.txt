🚀 NEXT STEPS TO GO LIVE:

1. Clone on production server:
   git clone https://github.com/axiscreatorhub/Cape-Ember.git
   cd Cape-Ember

2. Configure production environment:
   cp .env.production.example .env.production
   nano .env.production  # Fill in your credentials

3. Setup SSL certificate:
   mkdir -p ssl/certs
   sudo certbot certonly --standalone -d capeembercoffee.co.za
   sudo cp /etc/letsencrypt/live/capeembercoffee.co.za/fullchain.pem ssl/certs/cert.pem
   sudo cp /etc/letsencrypt/live/capeembercoffee.co.za/privkey.pem ssl/certs/key.pem

4. Deploy:
   chmod +x deploy.sh
   ./deploy.sh deploy --env production

5. Verify:
   curl https://capeembercoffee.co.za/health
