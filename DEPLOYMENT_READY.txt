🚀 QUICK START GUIDE:

1. LOCAL DEVELOPMENT
   $ docker-compose up -d
   $ open http://localhost:3000

2. PREPARE PRODUCTION SERVER
   $ ssh deploy@capeembercoffee.co.za
   $ mkdir -p /opt/cape-ember-coffee
   $ git clone https://github.com/capeembercoffee/capeembercoffee.co.za.git

3. CONFIGURE ENVIRONMENT
   $ cp .env.production.example .env.production
   $ nano .env.production  # Add your values

4. SETUP SSL
   $ mkdir -p ssl/certs
   $ sudo certbot certonly --standalone -d capeembercoffee.co.za

5. DEPLOY
   $ chmod +x deploy.sh
   $ ./deploy.sh deploy --env production

6. VERIFY
   $ curl https://capeembercoffee.co.za/health
